A closer look at some of the things I’ve built and what I focused on in each.

01

Attendance Management System (QR + Automation)

This system replaces slow manual roll calls with a simple digital flow. Students scan a unique QR code to mark their presence, and the system updates everything automatically.

It includes separate dashboards for admin, staff and students, real-time tracking and exportable reports for each class and subject.

Tech used

HTML, CSS, JavaScript, Django

What I did

  • Designed role-based flows for admin, staff and students.
  • Built QR-based session handling and basic validations.
  • Created clean attendance views and export options.
02

Rupasara – Image Universe Platform

Rupasara is an image universe inspired by platforms like Pexels and Pinterest. The goal is to keep browsing peaceful, with sharp thumbnails, simple filters and a focus on discovery.

I experimented with search, categories and layout decisions that make image exploration feel light instead of chaotic.

Tech used

Full-stack web (HTML, CSS, JS, Django), basic image handling

What I did

  • Crafted the core UI for browsing and searching images.
  • Thought through how creators and viewers interact with content.
  • Optimised for a clean, distraction-free look.
03

TaskFlow – Task Management UI

TaskFlow is a front-end-only task management interface created to practice UI states: empty views, loading states, filters and simple micro-interactions.

The focus was less on backend and more on how it feels to add, edit and complete tasks in a clean layout.

Tech used

HTML, CSS, JavaScript

What I did

  • Built a responsive dashboard-style layout.
  • Implemented task lists with basic filtering and state changes.
  • Experimented with subtle hover and focus animations.
